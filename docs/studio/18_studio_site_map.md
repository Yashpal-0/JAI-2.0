# 18 Studio Site Map

## Authenticated Application Routes
Every URL below requires an authenticated session (Bearer Token).

- `/dashboard`
  - High-level metric counters (Active Projects, Completed, Total Paid, Pending).
  - Recent activity feed.
  - "Government ID Required" alert banner.
- `/projects`
  - List of scoped and contracted development projects.
  - Status filters.
- `/contracts`
  - Review and sign legally binding service agreements.
- `/payments`
  - Track total and pending financial obligations.
  - Form to upload payment proof (Bank Transfer details provided).
- `/invoices`
  - Searchable list of issued invoices with PDF download capabilities.
- `/messages`
  - Threaded communication portal directly connecting the client with "Admin".
- `/notifications`
  - Chronological feed of project updates, payment confirmations, and system announcements.
- `/quotations`
  - Intake portal for new project requests (Budget, Timeline, Scope).
- `/scheduler`
  - Booking interface for Google Calendar video calls.
- `/settings`
  - Profile management (Avatar, Phone, Name).
  - Verification hub (Government ID).
- `/jai`
  - Full-page interface for the Zerostic AI Assistant.
